Principaux champs de données pour les comptes

Les champs énumérés ci-dessous peuvent paraître au formulaire des comptes, par ordre alphabétique. Les champs verrouillés lors de la validation de Numéro d'entreprise sont marqués d'un *. Votre organisation peut reconfigurer aisément les formulaires de comptes.

Données de base

  • Description du compte : Présente une description du compte.

  • Étiquettes du compte : Organisez avec une certaine souplesse et classez par catégorie les comptes en fonction d'une liste contrôlée des balises d'un compte. Utile lorsque les identifiants traditionnels (SCIAN, type de constitution en personne morale) ne suffisent pas.

  • Accréditation : Faites un suivi des licences, permis, certifications et autres accréditations de comptes.
  • Type de client SC : Indique si le compte a des activités actuelles ou passées financées par des subventions et contributions, comme a présenté une demande, a signé un accord de contribution, a agi en qualité de garant solidaire.
  • Principales personnes-ressources : principales personnes-ressources pour le compte. Peut se gérer avec des affiliations ou des relations ou des rôles.

  • Numéro de compte d'un ancien système : (facultatif) Si le compte a été migré d'un autre système, saisissez le numéro du compte utilisé dans ce système.

  • Nom : Faites un suivi du nom d'exploitation*, du nom légal* et de l'historique des noms légaux des comptes.

Données sur l'organisation

  • Adresses : Un compte peut comporter plusieurs adresses.

  • Affiliations : Une gamme d'options pour faire un suivi de la propriété d'entreprises, des membres des conseils d'administration et des fonctions, des parties affiliées et autres relations se trouvent dans les affiliations.

  • Identité des entreprises : Lorsque l'identité d'une entreprise est validée, tous les champs concernant l'identité de l'entreprise sont verrouillés.
  • Type d'entreprise* : Sélectionnez l'option appropriée. Les types peuvent être liés à une sélection à but lucratif / à but non lucratif.
  • Numéro d'entreprise de l'ARC*: Ce champ a été configuré pour faire en sorte que seuls les numéros d'entreprise en règle sont permis.
  • Renseignements financiers
    • Un compte renferme les données requises par le système financier, comme les identifiants d'un fournisseur ou d'un client.
    • Comptes bancaires : Saisissez les renseignements du compte bancaire (retrait ou dépôt). Ce domaine peut être à haute sécurité afin que seuls les membres du personnel autorisées puissent avoir accès aux données.

  • Données sur la constitution en personne morale : Faites un suivi des principaux renseignements concernant la constitution en personne morale.

  • Statut juridique : Choisissez l'une des options disponibles.
  • Codes SCIAN : il est possible d'attribuer à un compte un code SCIAN primaire et plusieurs codes SCIAN secondaires.
  • Date de début des activités : Saisissez la date à laquelle l'organisation a entamé ses activités.
  • Compte parent : Sélectionnez un compte parent (p. ex., dans le cas de comptes organisés hiérarchiquement ou de comptes parents / auxiliaires). Lisez davantage sur le sujet dans Gérer les comptes et les contacts.
  • Numéro d'entreprise provincial : Inscrivez, le cas échéant, le numéro d'entreprise provincial.
