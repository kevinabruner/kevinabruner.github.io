View project budget

A project budget is created automatically when a contribution agreement is generated for a project. The budget is based on the cost items reviewed during project assessment. The project budget displays the departmental funding allocated by cost category / subcategory.

To view a project budget: 

  1. Open a project and navigate to the project budget table.
    You can also view budget items in the project site map. However, that grid displays differently than the one displayed below.
  2. Note these fields on the budget are read-only: 
    • Major cost category: Cost categories defined during project assessment
    • Cost subcategory: (optional) sub-group of the major cost category
    • Cost item detail: (optional) brief description of the item or category
    • Rate of assistance: Percentage of the cost that is paid by the department
    • Departmental amount: Total departmental amount for the costs items in the given category
      (Note: cost items defined as "Budget element" display the departmental amount for individual items)
    • Expended amount: Amount that has been disbursed to date
    • Pending amount: Amount recommended for disbursement, but pending approval
    • Free balance: Departmental amount â€“ expended amount
    • Disb. + pending: % of available funds for a budget category, including pending amount

Note: After a project has been approved, changes to the budget can only be made using an amendment.
