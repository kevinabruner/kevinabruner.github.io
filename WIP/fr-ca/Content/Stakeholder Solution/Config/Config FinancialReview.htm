Configurer une revue financière

Le GPSC comprend une fonction dans laquelle un utilisateur peut réaliser une comparaison financière des états financiers du Client SC. La comparaison est basée sur l'exercice actuel, les deux exercices précédents et les deux exercices projetées. Les renseignements des états financiers peuvent être téléchargés à partir des données saisies sous le compte du Client SC ou entrés pendant la comparaison.

Les tâches clés pour configurer la revue financière:

  1. L'entité de revue financière a un formulaire séparé pour les états financiers des entités à but lucratif et sans but lucratif. Validez que chaque formulaire recueille les renseignements requis pour examiner les états financiers à votre organisation. Le formulaire possède cinq sections, des champs additionnels peuvent être ajoutés pour satisfaire aux besoins de votre organisation.
    Remarque: à l'exception des champs calculés, il n'y a aucune règle d'affaire dans le formulaire d'état financier.
  2. Les formulaires sont configurés afin de correspondre automatiquement au type de client examiné. Par exemple, le formulaire pour les organisations à but lucratif s'affiche lors d'une revue financière pour une organisation à but lucratif.
  3. Pour chacune des formulaires, vérifier si les options dans le champ Statut de la revue financière (egcs_financialreviewstatus) sont valides pour votre organisation. 
    Les options par défaut sont [Sous revue / Revue achevé].

    Remarque: le formulaire de revue financière est configuré pour afficher jusqu'à cinq états financiers dans un seul affichage. Tous les champs dans les formulaires d'état financier sont affichés dans le formulaire de revue financière.

Sections du formulaire de revue financière

Section

Description

Champs

Général

 Les renseignements liés à la période couverte par l'état. exercice financier; type d'état financier; date; ajusté; et satisfait les exigences de rapport.

Bilan

Les champs de données liés aux actifs, aux passifs et à l'équité.

Les règles d'affaire s'assurent que les actifs totaux = total des passifs + équité
(si ce n'est pas égal, un message d'erreur s'affiche)

Les totaux reflètent les données dans le formulaire. par exemple,

Total des passifs courants = Découvert bancaire + comptes créditeurs + autres passifs + partie actuelle LTD

Déclaration de revenus

Les champs de données liés à la déclaration de revenus

Certains champs sont calculés basés sur les renseignements entrés dans le formulaire.
