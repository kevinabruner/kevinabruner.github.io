Service Web SharePoint

Le Service Web SharePoint est le terme collectif utilisé pour décrire une série de services Web utilisés afin de soutenir le portail. Il est mis en oeuvre par le EGCS.Portal.WebService.SharePoint du projet. Ces services Web visent à soutenir les parties Web et Javascript côté client, y compris les parties Web SPFx.

  • Ces services Web sont directement hébergés par SharePoint.
  • Les services Web SharePoint sont RESTful.
  • Les méthodes de services sont appelées au moyen de HTTP POST.
  • Les paramètres des méthodes des services Web sont chiffrés au moyen de JSON.
  • Les résultats de retour des méthodes des services Web sont chiffrés au moyen de JSON.
  • Les services Web sont hébergés sur un chemin relatif /<site>/_layouts/15/EgcsPortal/<web service page>.
  • Le nom de la méthode pour les services Web est joint à l’URL du service Web, par ex. : /client/AC000922/_layouts/15/EgcsPortal/Account.svc/ApplicationStatusGet

L'exemple appelle la méthodeAccount.svc method ApplicationStatusGet dans la zone client AC000922.

  • Les services Web sont appelés dans le contexte de la sécurité de l'utilisateur de SharePoint. Le service Web fonctionne au moyen de l'identité de l'utilisateur.
