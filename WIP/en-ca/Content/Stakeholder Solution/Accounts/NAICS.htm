NAICS codes

To assign NAICS codes to an account:

  1. In an account record, open the Account Profile > NAICSsection.

  2. Click the lookup icon to choose an appropriate NAICS code.
  3. To select additional NAICS codes, use the Other grid and click the + icon.
    Alternatively, click the NAICS icon in the account sitemap.
  4. Save and close the record.

Notes: 

  • You can type the initial digits of the code, or words used in the name of the code to quickly find the appropriate code.
  • NAICS codes can be leveraged for reports and dashboards.
  • The North American Industry Classification System (NAICS) is a classification system used for organizing economic data by industry, and is used by Canada, Mexico and the United States.
  • GCPM is populated with the most recent version from Statistics Canada, Canada 2017 version 2.0. The next release is slated for 2022.
