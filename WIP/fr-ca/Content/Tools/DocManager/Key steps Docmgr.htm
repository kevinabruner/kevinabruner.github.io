Principales étapes pour la configuration du gestionnaire de documents

Avant de configurer le gestionnaire de documents, voici les étapes à suivre :

  • Prérequis : décidez de la structure de documents qui convient à votre organisation. Le cadre de gestion des documents peut fonctionner avec n’importe quel système de gestion des documents (GCDocs, SharePoint, etc.); il n’y a pas d’instructions particulières aux SGD, seulement quelques conseils utiles.
    Consulter Configurer un système de gestion des documents.
  • Configuration préalable : configurez la solution de gestion des documents dans CRM avant de configurer le cadre de gestion des documents.
    Consulter Paramètres des solutions pour le gestionnaire de documents et Configurer les noms d'utilisateur pour le système de gestion de documents

Pour configurer la solution de gestion des documents :

  1. Configurez le site de documents et les emplacements de dossiers prédéfinis.
    ConsulterConfigurer les lieux de documents.
  2. Établissez la structure des dossiers :
    1. Permettez la gestion de documents pour une entité en particulier ou plus. Cela associe un ensemble d’emplacements de documents à une entité donnée.
      Par exemple, configurez les documents associés aux comptes à stocker dans un dossier de compte, avec des sous-dossiers pour les documents juridiques et les états financiers.
      Consultez Configurer les entités du document
    2. Créez des emplacements de dossiers et de sous-dossiers pour l’entité. Remarque : Il s’agit d’une activité unique visant à établir une structure de dossiers hiérarchique.
      Consulter Configurer les lieux de documents.
    3. (Facultatif) Ajoutez des métadonnées pour tous les documents de ce type d’entité. Remarque : Ce n’est pas actuellement mis en œuvre. ConsulterConfigurer les champs de documents
  3. Configurez les types de documents.
    Voir Configurer les types de documents
