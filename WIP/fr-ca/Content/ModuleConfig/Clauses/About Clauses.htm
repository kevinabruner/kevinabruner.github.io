À propos des clauses

Une entente de contribution (EC) établi les obligation du gouvernement du Canada et d'un bénéficiaire en ce qui concerne un paiement de transfert ou plus. GPSC offre deux options pour gérer les clauses de l'EC.

Modèles de document
  • Les clauses standards et les clauses avec des variations sont gérés dans les modèles de document.
  • Les clauses peuvent être insérer dans une EC selon le client ou l'évaluation du projet.
  • Les clauses gérées directement dans le modèle de document peuvent être automatisées ou formatées selon les données dans le GPSC.
  • Par exemple, une clause liées aux obligations juridiques d'un bénéficiaire en vertu de la Loi sur les langues officielles (LLO) a trois options. Toutes les options sont comprises dans le modèle de document. Selon l'évaluation de l'agent des exigences de la LLO, la clause appropriée est insérées dans l'EC.
  • La programmation XML est requise pour configurer les modèles de document.
Fonctionnalité de la clause
  • La souplesse gère les clauses particulières à un projet (conditions spéciales) dans une entente de contribution de projet.
  • Un modèle de clause de programme contient des clauses particulières au programme qui sont copiées à chaque projet dans le programme en utilisant le plugiciel clone.
  • Les agents peuvent être en mesure de réviser, de modifier ou d'enlever les clauses qui sont insérées dans l'EC pour le projet selon les paramètres.
  • Un modèle peut être utilisé dans de multiples programmes ou volets.

  • Les clauses peuvent être configurées afin d'être:

    • Modifiable par un agent
    • Inclues ou exclues de manière sélective pour un projet donné.
    • Modifiées sur les critères particuliers au projet.

    • Remarque, les nouvelles clauses crées manuellement par un agent ne sont pas disponibles aux autres projets ou aux autres utilisateurs sur le système.

  • Les clauses peuvent aussi être configurées pour afficher dans un format approprié et une hiérarchie (clause, sous-disposition, alinéa, élément, sous-alinéa) afin de configurer de manière appropriée le format de numérotation utilisée dans l'entente de contribution.
  • Les modèles de clause gèrent la majorité des clauses de l'EC, par contre, cela requiert un niveau complexe de script XML.

Les conditions spéciales sont des clauses particulières au projet.

Le GPSC comprend un exemple de document d'EC avec deux modèles de clause uniques sous différents programmes (SFE - Fonds d'entrepreneuriat et le programme de base du GPSC). Dans l'échantillon EC du GPSC, les clauses spéciales sont insérées dans la section des conditions spéciales.
