Standard Fields (Text, Number)

 

 

This section details how to use SetFieldValue, GetFieldValue, GetRowData, and Custom Field context.Data with standard CRM data types.

 

 

Single Line of Text

 

GetFieldValue

 

This will retrieve the value of a textbox column for the first row.  The value will be a string representing the contents of the textbox, regardless of the 'Unformatted' argument specified.

 

 

 

SetFieldValue

 

This will set the value of a textbox column for the first row.  Expected input is a string.

 

 

 

GetRowData

 

This will retrieve the value of a textbox column from all rows in the EditAble CRM Grid. The value will be a string representing the contents of the textbox.

 

 

 

Custom Field Context

 

This will retrieve the value of a textbox column from the Custom Field context argument. The value of will be a string representing the contents of the textbox for both context.Data and context.FormattedData.

 

 

 

 

Multiple Lines of Text

 

GetFieldValue

 

This will retrieve the value of a memo column for the first row.  The value will be a string representing the contents of the memo field, regardless of the 'Unformatted' argument specified.  The contents may contain encoding e.g. '\n' representing a new line.

 

 

 

SetFieldValue

 

This will set the value of a memo field for the first row.  Expected input is a string. '\n' denotes new line.

 

 

 

GetRowData

 

This will retrieve the value of a memo column from all rows in the EditAble CRM Grid. The value will be a string representing the contents of the memo field.

 

 

 

Custom Field Context

 

This will retrieve the value of a memo column from the Custom Field context argument. The value of will be a string representing the contents of the memo field for both context.Data and context.FormattedData.

 

 

 

 

Two Options - Checkbox

 

GetFieldValue 

 

This will retrieve the value of a checkbox column for the first row.  The value will be a boolean (either true or false) that represents its current checked state, when 'Unformatted' unspecified or false.

 

 

The value will be a string representation of the label when the 'Unformatted' argument is specified true.

 

 

 

SetFieldValue

 

This will set the value of a two option checkbox field for the first row.  Expected input is a boolean (either true or false).

 

 

 

GetRowData

 

This will retrieve the value of a checkbox column from all rows in the EditAble CRM Grid. The value will be a boolean (either true or false) that represents its current checked state.

 

 

 

Custom Field Context

 

This will retrieve the value of a checkbox column from the Custom Field context argument. The context.Data value will be a boolean representing the checked state and the context.FormattedData value will be a string representation of the label.

 

 

 

Two Options - Option Set (Picklist)

 

GetFieldValue

 

This will retrieve the value of a two option picklist column for the first row.  The value will be a boolean (either true or false) that represents its current set state when the 'Unformatted' argument is unspecified or false.

 

 

The value will be a string representation of the label when the 'Unformatted' argument is specified true.

 

 

 

SetFieldValue 

 

This will set the value of a two option picklist column for the first row.  Expected input is a boolean (either true or false).

 

 

 

GetRowData

 

This will retrieve the value of a two option picklist column from all rows in the EditAble CRM Grid. The value will be a boolean (either true or false) that represents its current set state.

 

 

 

Custom Field Context

 

This will retrieve the value of a checkbox column from the Custom Field context argument. The context.Data value will be a boolean representing the checked state and the context.FormattedData value will be a string representation of the label.

 

 

 

 

Whole Number

 

GetFieldValue

 

This will retrieve the value of a whole number (integer) column for the first row.  The value will be a string representation of the integer when the 'Unformatted' argument is unspecified or false.

 

 

The value will be an integer when the 'Unformatted' argument is specified true.

 

 

 

SetFieldValue

 

This will set the value of a whole number (integer) column for the first row.  Expected input is an integer.  Floating point values will be rounded to the nearest integer.

 

 

 

GetRowData

 

This will retrieve the value of a whole number (integer) column from all rows in the EditAble CRM Grid. The value will be an integer.

 

 

 

Custom Field Context

 

This will retrieve the value of a whole number column from the Custom Field context argument. The context.Data value will be an integer and the context.FormattedData value will be a string representation of the integer.

 

 

 

 

Decimal Number

 

GetFieldValue

 

This will retrieve the value of a decimal column for the first row.  The value will be a string representation of the decimal number when the 'Unformatted' argument is unspecified or false.

 

 

The value will be a decimal value when the 'Unformatted' argument is specified true. 

 

 

SetFieldValue

 

This will set the value of a decimal column for the first row.  Expected input is a decimal number.  Values will be rounded to the field specified precision.

 

 

 

GetRowData

 

This will retrieve the value of a decimal column from all rows in the EditAble CRM Grid. The value will be an decimal number.

 

 

 

 

Custom Field Context

 

This will retrieve the value of a decimal number column from the Custom Field context argument. The context.Data value will be a decimal value and the context.FormattedData value will be a string representation of the decimal number.

 

 

 

 

Floating Point Number

 

GetFieldValue

 

This will retrieve the value of a floating point number column for the first row.  The value will be a string representation of the floating point number when the 'Unformatted' argument is unspecified or false.

 

 

The value will be a floating point number when the 'Unformatted' argument is specified true.

 

 

SetFieldValue

 

This will set the value of a floating point number column for the first row.  Expected input is a floating point number.  Values will be rounded to the field specified precision.

 

 

 

GetRowData

 

This will retrieve the value of a decimal column from all rows in the EditAble CRM Grid. The value will be an decimal / floating point number.

 

 

 

 

Custom Field Context

 

This will retrieve the value of a floating point number column from the Custom Field context argument. The context.Data value will be a floating point / decimal value and the context.FormattedData value will be a string representation of the floating point number.

 

 

 

 

Currency / Money

 

GetFieldValue

 

This will retrieve the value of a currency column for the first row.  The value will be a string representation of the currency when the 'Unformatted' argument is unspecified or false.

 

 

The value will be a decimal value when the 'Unformatted' argument is specified true.

 

 

 

 

SetFieldValue

 

This will set the value of a currency column for the first row.  Expected input is a floating point number or an object with the floating point number property Value.  Values will be rounded to the field specified precision.

 

 

 

GetRowData

 

This will retrieve the value of a currency column from all rows in the EditAble CRM Grid. The value will be an decimal value.

 

 

 

Custom Field Context

 

This will retrieve the value of a currency column from the Custom Field context argument. The context.Data value will be a floating point / decimal value and the context.FormattedData value will be a string representation of the currency.
