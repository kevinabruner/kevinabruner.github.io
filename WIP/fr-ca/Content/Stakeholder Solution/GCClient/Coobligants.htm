Codébiteurs

Les codébiteurs sont des particuliers ou des organisations qui assument la responsabilité de respecter les modalités de l'accord de contribution (en plus de SC client), y compris le remboursement de la contribution, s'il y a lieu. Les termes coclient, cobénéficiaire, coemprunteur et codemandeur sont parfois utilisés afin de désigner un codébiteur.

Si un codébiteur a été identifié pendant l'évaluation de projet, il faut exercer le même niveau de diligence raisonnable pour le codébiteur que pour le client.

Pour établir un codébiteur pour un projet :

  1. Dans un enregistrement de projet, naviguer à Vérification des états financiers et cliquez sur +Nouveau afin d'ouvrir un nouvel enregistrement pur le codébiteur.
  2. Assurez-vous que les information dans les détails du projet sont exacts.
  3. Dans la section codébiteurs, saisissez l'information requise :
    • Codébiteurs : vérification de clients SC
    • Type de codébiteurs : sélectionnez le type de codébiteur pour ce projet.
    • Exposition du client : saisissez la valeur en dollars de l'exposition du codébiteur au ministère
    • Brève description de l'implication du projet : fournissez une description de la participation du codébiteur dans ce projet.
    • Commentaire d'agent / conseiller : commentaires fournis par l'agent
  4. Sauvegardez l'enregistrement et fermez le formulaire. L'information sera remplie dans la grille.

Dans l'entité Client SC, une grille en lecture seule affiche tous les projets dans lesquels le client est codébiteur. L'information offre un instantané rapide au moment de faire la diligence raisonnable et le suivi du client.
