TypeScript

Des renseignements sur le TypeScript se trouve sur son site Web officiel :https://www.typescriptlang.org/

Certaines des principales différences avec JavaScript sont énumérées ici.

Syntaxe de TypeSript

Types statiques

L'une des éléments les plus importants lors du travail avec TypeScript est de toujours déclarer le type de la variable.

Type

Remarque:
Types de déclaration

var n: number = 5;

function add (a: number, b: number): number { return a + b; }
Types multiples

function(shape: Square | Circle): void { }
Type dynamique

Déclarer le type puisque « toutes » accepte toutes les valeurs.
Type de typographie

Syntax: <string>variableName or variableName as string

Interfaces

Il y a une syntaxe particulière à TypeScript qui est similaire aux interfaces dans Java/C#.

Les interfaces sont utilisées pour créer des objets fortement typés ou fortement statiques.

Les interfaces dans TypeScript sont seulement utilisé pour offrir une vérification du type fort au moment de la compilation et ne possèdent aucun sens d'héritage dans JavaScript.

Espace de nommage par rapport à module

C'est une distinction particulière à TypeScript.

Les espaces de nommage se situent dans l'espace de nommage général comme variables générales.

Les modules sont des fichiers JS indépendants qui sont téléchargé dans un autre module et qui se situent dans l'espace de nommage local.

Énumérations

Contrairement aux autres langues, les énumérations peuvent être numériques ou des fils.

Il y a deux types d'énumération

  • Énumération régulière
  • Elles sont transpilées dans des libellés d'objet réguliers dans le fichier JS.
  • Énumération constante
  • Elles ne sont pas transpilée dans le fichier JS; elles sont considérées comme des constantes de fil dans C# lesquelles remplacent toutes les références de la constante de fil avec le fil actuel lors de la compilation.

Installation de l'ensemble TypeScript

Afin de travailler avec TypeScript dans Visual Studio, les ensembles @types/xrm and @types/jquery doivent être installés. Pour les installer, suivez ces étapes :

  1. Installez le Programme d'installation d'ensemble dans Visual Studio
  2. Installez les ensembles @types/xrm and @types/jquery
  3. Faites un clic droit sur le projet et choisissez Quick Install Package (Installation rapide de l’ensemble)
  4. Installez les ensembles.
  5. Inclure le dossier node_modules dans le projet.

    Si le dossier node_modules n'est pas visible, utilisez « Afficher tous les fichiers ». Faites un clic droit sur le dossier node_modules et l'inclure dans le projet.
